gitbook的使用

gitbook的使用

文档地址

安装

$ sudo npm install -g gitbook-cli

命令

# 初始化项目|生成目录文件
$ gitbook init
# 本地预览
$ gitbook serve
# 建立静态网站
$ gitbook build

book.json

{
  "plugins": ["codeblock-filename", "disqus"],
  "pluginsConfig": {
    "disqus": {
      "shortName": "blog-honghaitao-net"
    }
  }
}
