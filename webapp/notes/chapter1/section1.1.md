# gitbook的使用
[文档地址](https://github.com/GitbookIO/gitbook/blob/master/docs/setup.md)
## 安装
```bash
$ sudo npm install -g gitbook-cli
```
## 命令
```bash
# 初始化项目|生成目录文件
$ gitbook init
# 本地预览
$ gitbook serve
# 建立静态网站
$ gitbook build
```
